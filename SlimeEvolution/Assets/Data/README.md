# 初始数据说明

## 在 Unity Editor 中创建 ScriptableObject

1. 右键 Assets/Data/SlimeData → Create → SlimeEvolution → SlimeData
2. 右键 Assets/Data/TraitData → Create → SlimeEvolution → TraitData
3. 右键 Assets/Data/SkillData → Create → SlimeEvolution → SkillData

## JSON 文件

JSON 文件定义了 3 种初始史莱姆、1 种特性和 3 种技能的数值参考。
在 Unity Editor 中创建 ScriptableObject 后，按 JSON 中的值配置即可。
