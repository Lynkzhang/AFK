# 史莱姆进化 — 玩家指南

## 游戏简介

史莱姆进化是一款放置养成 + 回合制战斗游戏。你管理一座史莱姆牧场，让史莱姆自动分裂繁殖、变异进化，筛选出最强的个体组队挑战关卡副本，赢取金币和晶石来升级设施、购买道具，不断循环强化你的史莱姆军团。

## 启动游戏

1. 打开浏览器访问游戏页面（开发模式下为 `http://localhost:5173`）
2. 游戏自动加载并显示主界面
3. 场景中可以看到你的初始史莱姆在牧场中活动

---

## 核心玩法循环

```
观察史莱姆自动分裂
    ↓
筛选 → 剔除/出售弱个体
    ↓
封存精英个体（保护不被误删）
    ↓
组队 → 挑战战斗关卡
    ↓
赢取金币 💰 和晶石 💎
    ↓
升级设施 / 购买道具
    ↓
循环 → 更快分裂 → 更强变异 → 更难关卡
```

## 史莱姆系统

### 自动分裂

- 史莱姆会定期自动分裂，产生新的后代
- 分裂间隔默认 10 秒，可通过升级「繁殖加速器」缩短
- 牧场满员时停止分裂

### 稀有度

从低到高共 5 个等级：

| 稀有度 | 颜色 |
|--------|------|
| Common（普通） | 🟢 绿色 |
| Uncommon（非凡） | 🔵 蓝色 |
| Rare（稀有） | 🟣 紫色 |
| Epic（史诗） | 🔴 红色 |
| Legendary（传说） | 🟡 金色 |

### 属性

每只史莱姆有以下属性：

| 属性 | 说明 |
|------|------|
| **Health（生命）** | 战斗中的生命值 |
| **Attack（攻击）** | 影响普通攻击和技能伤害 |
| **Defense（防御）** | 减少受到的伤害 |
| **Speed（速度）** | 决定战斗中的行动顺序 |

### 变异

分裂时后代可能发生变异：
- 属性随机浮动
- 可能获得新的特性（Trait）
- 可能学会新的技能（Skill）
- 升级「变异实验室」可提高变异率

### 史莱姆操作

每只史莱姆卡片上有三个按钮：

| 按钮 | 功能 |
|------|------|
| **剔除** | 直接移除该史莱姆（不获得金币） |
| **出售** | 出售该史莱姆获得金币（价格由稀有度、属性、特性、技能决定） |
| **封存** | 将该史莱姆移入封存库保护起来 |

## 封存系统 📦

封存库用于保护你精心培育的精英史莱姆：

- 封存后的史莱姆不会出现在牧场列表中，不会被误删
- 封存容量初始为 10，可通过升级「封存扩展」增加（每级 +5）
- 封存的史莱姆可以随时取回到牧场
- 封存的史莱姆也可以直接从封存库中移除

### 封存库操作

- 点击主界面的 **📦 封存库** 按钮打开
- 在封存库中可以查看所有封存的史莱姆
- 每只封存史莱姆可以选择 **取回**（回到牧场）或 **移除**

## 战斗系统 ⚔

### 章节与关卡

游戏共有 **3 章 30 关**：

| 章节 | 名称 | 关卡 | 解锁条件 |
|------|------|------|----------|
| **第 1 章** | 史莱姆平原 | 1-1 到 1-10 | 默认解锁 |
| **第 2 章** | 火焰洞窟 | 2-1 到 2-10 | 通关 1-10 |
| **第 3 章** | 寒冰深渊 | 3-1 到 3-10 | 通关 2-10 |

每章第 5 关为**精英关**，第 10 关为 **BOSS 关**。

### 战斗流程

1. 点击 **⚔ 战斗** 按钮
2. 选择章节（已解锁的章节可选）
3. 选择关卡
4. 选择出战的史莱姆（最多 4 只）
5. 战斗自动进行（回合制）
6. 胜利后获得奖励

### 战斗机制

- **回合制**：按速度（Speed）决定行动顺序
- **技能系统**：史莱姆和敌人各有技能，有冷却时间
- **攻击类型**：单体攻击（single）和群体攻击（aoe）
- **防御技能**：可以提升防御力

### 战斗评价（星级）

每关胜利后根据表现获得 1~3 星评价：
- ⭐ 通关
- ⭐⭐ 较好表现
- ⭐⭐⭐ 完美通关

### 战斗奖励

| 奖励类型 | 说明 |
|----------|------|
| **金币** | 用于升级设施和购买道具 |
| **晶石** | 稀有货币，用于购买高级道具 |
| **道具** | 部分关卡额外掉落道具（如变异催化剂） |

## 设施系统 🏗

通过升级设施来增强整体能力。所有设施最高 10 级，升级需要金币。

| 设施 | 效果 | 初始值 |
|------|------|--------|
| **繁殖加速器** | 缩短分裂间隔（每级约 -10%） | 10 秒 |
| **场地扩展** | 增加牧场容量（每级 +2） | 12 |
| **封存扩展** | 增加封存库容量（每级 +5） | 10 |
| **变异实验室** | 提高变异率（每级 +15%） | ×1.0 |

### 升级费用

升级费用随等级递增：`基础费用 × 当前等级 × 1.5`

---

## 商店系统 🛒

商店提供三种道具，可用金币或晶石购买：

| 道具 | 价格 | 效果 |
|------|------|------|
| **变异催化剂** | 200 金币 | 下次分裂变异概率 ×2（一次性） |
| **属性强化剂** | 300 金币 | 对指定史莱姆随机属性 +3~5 |
| **稀有精华** | 50 晶石 | 下次分裂稀有特性概率 ×3（一次性） |

### 道具使用

购买的道具进入背包，需要手动使用：
- **变异催化剂** / **稀有精华**：使用后在下次分裂时自动生效
- **属性强化剂**：使用时需要指定一只目标史莱姆

---

## 存档系统

- 点击 **保存** 按钮将当前进度保存到浏览器本地存储
- 点击 **加载** 按钮读取上次保存的进度
- 存档包含：所有史莱姆、金币、晶石、关卡进度、设施等级、道具背包、封存库
- ⚠️ 清除浏览器数据会丢失存档

## 新手攻略建议

1. **前期**：让史莱姆自由分裂，观察变异结果
2. **筛选**：出售低稀有度、低属性的史莱姆换取金币
3. **封存**：遇到高稀有度或高属性的精英，立刻封存保护
4. **战斗**：用最强的 4 只史莱姆挑战关卡
5. **升级**：优先升级「繁殖加速器」和「场地扩展」
6. **商店**：攒够晶石后购买「稀有精华」提高精英出现率
7. **循环**：不断分裂→筛选→战斗→升级，逐渐推进更高章节

## GM 命令（开发者模式）

> 按 F12 打开浏览器控制台，输入 `__GM.命令()` 即可使用。

常用命令：

```js
__GM.addSlime()              // 添加一只史莱姆
__GM.setCurrency(9999)       // 设置金币为 9999
__GM.setCrystal(999)         // 设置晶石为 999
__GM.triggerSplit()          // 立即触发一次分裂
__GM.startBattle('1-1')      // 开始 1-1 关卡战斗
__GM.unlockChapter(3)        // 解锁全部章节
__GM.upgradeFacility('breeding-accelerator')  // 升级繁殖加速器
__GM.getState()              // 查看当前完整游戏状态
```

完整命令列表请参阅 [README.md](./README.md#gm-命令开发者模式)。
