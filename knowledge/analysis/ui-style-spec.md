# UI 视觉规范 — 史莱姆进化

> 版本 1.0 | 2025-04 | 作者: yue (主美)

## 1. UI 设计总则

**风格关键词**：圆角卡片式、半透明毛玻璃、明亮配色、可爱但不幼稚

**参考方向**：
- *原神* — 卡片式信息面板 + 毛玻璃背景
- *Slime Rancher 2* — 圆润图标和按钮风格
- *Slay the Spire* — 战斗界面信息布局（行动条、技能区清晰分离）

**设计原则**：
1. **信息层级清晰**：一眼看到最重要的信息，次要信息不抢视觉
2. **操作反馈明确**：所有可交互元素都有 hover/press/disabled 状态
3. **一致性**：全游戏使用统一组件库，不做一次性特殊 UI
4. **适配**：基准分辨率 1920×1080，支持 16:9 和 16:10，UI 使用 Canvas Scaler（Scale With Screen Size）

## 2. 基础设计系统

### 2.1 栅格系统

- **基准分辨率**：1920 × 1080
- **安全边距**：上下左右各 40px
- **基础间距单位**：8px（所有间距为 8 的倍数：8, 16, 24, 32, 48, 64）
- **卡片间距**：16px
- **面板内边距**：24px

## 3. 组件规范

### 3.1 按钮

#### 主按钮 (Primary)
```
尺寸：最小 160×48px，内边距 24×12px
圆角：12px
背景：#5EEDB8（正常）→ #4AC89A（按下）→ #C0C8CC（禁用）
文字：#FFFFFF，22px Bold
阴影：0 2px 8px rgba(94,237,184,0.3)
动画：按下时 Scale 0.95，松开弹回 1.0（120ms Ease-Out）
```

#### 危险按钮 (Danger)
```
尺寸：同主按钮
圆角：12px
背景：#FF6B6B → #E05555（按下）
文字：#FFFFFF，22px Bold
```

#### 次要按钮 (Secondary)
```
尺寸：同主按钮
圆角：12px
背景：#F0ECE8 → #E0DCD8（按下）
文字：#6B7280，22px Regular
无阴影
```

#### 图标按钮 (Icon)
```
尺寸：48×48px（小）/ 64×64px（中）/ 80×80px（大）
圆角：50%（圆形）
背景：#FFFFFF Alpha 0.8
图标：24×24 / 32×32 / 40×40px，颜色 #2D3436
Hover：背景变为 #5EEDB8 Alpha 0.15
```

### 3.2 卡片面板

```
圆角：16px
背景：#FFFFFF Alpha 0.92，开启 Blur（半径 10px）
描边：1px #E8E0D8
内边距：24px
阴影：0 4px 16px rgba(0,0,0,0.08)
```

#### 史莱姆信息卡片
```
尺寸：240×320px（纵向）或 400×180px（横向列表项）
顶部区域（60%）：史莱姆 3D 预览或图标
底部区域（40%）：名称 + 稀有度标签 + 主属性摘要
稀有度标签：圆角 8px 色条，颜色见稀有度色表
```

### 3.3 进度条 / 血量条

```
外框：圆角 6px，高度 12px（小）/ 20px（大），背景 #E8E0D8
填充：圆角 6px，颜色按用途区分
  - HP：#6BCB77（>50%）→ #FFB347（25%–50%）→ #FF6B6B（<25%）
  - 经验/进度：#5EEDB8
  - 能量条：#B389F4
动画：值变化时 200ms 平滑过渡
```

### 3.4 列表项 / 选择行

```
高度：64px
内边距：16px 左右
背景：透明（默认）→ #5EEDB8 Alpha 0.10（hover）→ #5EEDB8 Alpha 0.20（选中）
左侧：48×48 图标/头像区
中间：名称（H3）+ 说明（Caption）
右侧：操作按钮或数值
分割线：1px #F0ECE8（仅相邻项之间）
```

### 3.5 弹窗 / 对话框

```
遮罩：全屏 #000000 Alpha 0.45
面板：居中，最大 640×480px，圆角 20px
标题：H2 居中
内容：Body 居中或左对齐
按钮区：底部 24px 边距，按钮间距 16px，水平居中
出现动画：Scale 0.9→1.0 + Opacity 0→1（200ms Ease-Out）
```

### 3.6 标签 / Badge

```
圆角：8px（方）或 50%（圆）
内边距：8×4px
字号：14px Bold
用途及颜色：
  - 稀有度标签：使用稀有度对应色，白色文字
  - 数量 Badge：#FF6B6B 背景，白色文字，圆形
  - 状态标签：#5EEDB8（正面）/ #FFB347（注意）/ #FF6B6B（负面）
```

### 3.7 Toast 提示

```
位置：屏幕顶部居中，距顶 80px
尺寸：自适应内容，最大 600px 宽
圆角：12px
背景：#2D3436 Alpha 0.88
文字：#FFFFFF，Body 字号
动画：从上方滑入（-40px → 0px，200ms），停留 2s，淡出（300ms）
```

## 4. 图标规范

- **风格**：填充式（Filled），线条粗细 2px 等效，圆角端点

## 6. 交互动效规范

| 交互 | 动效 | 时长 | 缓动 |
|------|------|------|------|
| 按钮按下 | Scale 1.0 → 0.95 | 80ms | Ease-Out |
| 按钮松开 | Scale 0.95 → 1.0 | 120ms | Ease-Out-Elastic |
| 面板打开 | Scale 0.9 → 1.0 + Opacity 0 → 1 | 200ms | Ease-Out |
| 面板关闭 | Scale 1.0 → 0.95 + Opacity 1 → 0 | 150ms | Ease-In |
| 列表项 Hover | 背景色淡入 | 100ms | Linear |
| 页面切换 | 旧页面左滑出 + 新页面右滑入 | 250ms | Ease-In-Out |
| 伤害数字弹出 | Scale 0 → 1.2 → 1.0 + 上浮 40px + 淡出 | 800ms | Ease-Out |
| 治疗数字弹出 | 同上，绿色 | 800ms | Ease-Out |
| 货币变化 | 数字滚动（Counting Up/Down） | 400ms | Ease-Out |
| Toast 出现 | 从顶部滑入 40px | 200ms | Ease-Out |
| Toast 消失 | Opacity 1 → 0 | 300ms | Ease-In |

## 7. 响应式适配

| 分辨率 | 处理方式 |
|--------|---------|
| 1920×1080 (基准) | 100% 缩放 |
| 2560×1440 | 等比放大，间距不变 |
| 1280×720 | 等比缩小，最小字号不低于 12px |
| 16:10 (1920×1200) | 上下留白区域增加，内容居中 |
| 非标比例 | 安全区域内 UI 保持完整，超出部分用背景填充 |

Canvas Scaler 参数：
```
UI Scale Mode: Scale With Screen Size
Reference Resolution: 1920 × 1080
Screen Match Mode: Match Width Or Height
Match: 0.5
```
