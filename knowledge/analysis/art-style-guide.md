# 美术风格指南 — 史莱姆进化

> 版本 1.0 | 2025-04 | 作者: yue (主美)

## 1. 总体风格定义

**风格关键词**：3D Low-Poly、可爱圆润、明亮饱和、半透明感、童话奇幻

**参考方向**：
- *Slime Rancher* — 史莱姆的圆润感和透明质感
- *Crossy Road / Monument Valley* — Low-Poly 的面数控制和色彩运用
- *Genshin Impact（提瓦特大世界）* — 明亮饱和的自然环境色彩方案
- *Pokémon Legends: Arceus* — Low-Poly 地形处理手法

**核心审美原则**：
1. **简洁但不简陋**：模型面数低，但轮廓必须可读、可爱
2. **色彩先行**：靠颜色区分，不依赖细节贴图
3. **半透明是标志**：史莱姆必须有通透感，这是本作视觉识别度的核心
4. **圆润 vs 棱角**：史莱姆本体全用圆滑曲线；环境/建筑用 Low-Poly 硬边面；两者对比形成视觉节奏

### 2.2 色彩使用规则

- **60-30-10**：画面面积 60% 为中性浅/底色，30% 为主色/场景色，10% 为强调色
- 稀有度用色见下方史莱姆文档
- **避免纯黑 `#000000` 和纯白 `#FFFFFF`**，始终使用上表中的墨夜和奶白替代
- UI 文字默认 `#2D3436`，次要文字 `#8D9A9E`

## 3. 模型规范

### 3.1 面数预算

| 类别 | 目标三角面数 | 硬上限 |
|------|-------------|--------|
| 史莱姆（基础形态） | 300–500 tris | 800 tris |
| 史莱姆（复杂变异形态） | 500–1000 tris | 1500 tris |
| 史莱姆附加部件（角、翅膀等） | 50–200 tris/件 | 300 tris/件|
| 培养场地（单块地板） | 50–100 tris | 150 tris |
| 场地环境物件（树、石、建筑） | 200–600 tris | 1000 tris |
| 饰品（装备在史莱姆上） | 50–150 tris | 250 tris |
| UI 3D 图标 | 100–300 tris | 500 tris |

### 3.2 建模规则

- **史莱姆主体**：使用 Subdivision Surface / Smooth Shading，保持圆润外形。不使用硬边
- **环境模型**：使用 Flat Shading（面着色），保持 Low-Poly 棱角感
- **尺寸统一**：史莱姆基础尺寸为 Unity 单位 1.0 × 1.0 × 1.0（宽 × 高 × 深），变异体按比例缩放，最大不超过 2.0
- **坐标原点**：模型底部中心
- **朝向**：默认面朝 Z+ 轴

### 3.3 拓扑注意

- 避免 N-gon，三角化后导出
- 史莱姆需预留骨骼绑定：至少 body（弹跳）、eye_L、eye_R（眨眼表情）
- 环境物件不需要骨骼

## 4. 材质与着色器

### 4.1 史莱姆材质

| 属性 | 值 |
|------|-----|
| 着色器 | Unity URP Lit (Transparent) |
| 渲染模式 | Transparent |
| Base Color Alpha | 0.55–0.75（半透明感核心） |
| Smoothness | 0.85–0.95（表面光泽） |
| Metallic | 0.0（非金属） |
| 次表面散射 (SSS) | 如引擎支持，启用浅层 SSS 增加通透感 |
| 边缘发光 (Fresnel) | 启用，强度 0.3–0.5，颜色为基础色提亮 20% |
| 法线贴图 | 不使用 |
| 纹理 | 不使用详细纹理，仅用 Vertex Color 或单色 |

> 核心视觉：史莱姆要看起来像果冻/琥珀——通透、有内部散射感、边缘比中心亮

### 4.2 环境材质

| 属性 | 值 |
|------|-----|
| 着色器 | Unity URP Lit (Opaque) |
| 渲染模式 | Opaque |
| Smoothness | 0.1–0.3（哑光） |
| 纹理 | 不使用贴图，全部用 Vertex Color 上色 |
| 法线贴图 | 不使用 |

### 4.3 特效材质

| 效果 | 实现方式 |
|------|---------|
| 分裂动画 | 粒子 + 模型变形（Blend Shape） |
| 变异高光 | 全身发光 Emission 闪烁 0.5s |
| 战斗技能 | Particle System，每种技能独立颜色主题 |
| UI 点击反馈 | 缩放弹性动画（Scale Tween） |

## 5. 光照方案

### 5.1 全局光照

| 参数 | 值 |
|------|-----|
| 方向光颜色 | `#FFF4E0` (暖白) |
| 方向光强度 | 1.2 |
| 方向光角度 | X: 50°, Y: -30° |
| 环境光模式 | Gradient |
| 天空色 | `#D4E8FF` (浅蓝) |
| 赤道色 | `#FFF8F0` (奶白) |
| 地面色 | `#E8DDD0` (暖米) |
| 环境光强度 | 0.8 |
| 阴影 | Soft Shadow，距离 30 单位 |
| 阴影颜色 | 不使用纯黑，场景阴影偏向 `#B0A8C0` (淡紫灰) |

### 5.2 场地差异化光照

不同培养场地通过调整光照和颜色滤镜区分氛围：

| 场地 | 方向光色偏移 | 环境色调 | 后处理 |
|------|-------------|---------|--------|
| 丰饶草地（默认） | 暖白不变 | 绿调 | 无 |
| 火焰之地 | +橙红 `#FFD4A0` | 暖橙 | 轻微 Bloom |
| 寒冰洞穴 | +冷蓝 `#C8E8FF` | 蓝白 | 轻微色温降低 |
| 神秘森林 | +紫绿 `#D0FFE0` | 深绿 | 轻微 Vignette |

### 5.3 战斗场景光照

- 光照稍强（强度 1.4），增加戏剧感
- 聚光灯追踪当前行动单位，强度 0.5，柔边
- 背景环境适度压暗（环境光强度降至 0.5）

## 7. 动画规范

### 7.1 史莱姆基础动画

| 动画名 | 描述 | 帧数 | 循环 |
|--------|------|------|------|
| idle | 轻微上下弹跳 + 呼吸感缩放 | 60 帧 (2s @ 30fps) | ✅ |
| hop | 短距离弹跳移动 | 20 帧 | ❌ |
| split | 身体拉伸→分裂为二 | 45 帧 | ❌ |
| hit | 被击中压扁→回弹 | 15 帧 | ❌ |
| attack | 前冲→回弹 | 20 帧 | ❌ |
| happy | 跳跃+旋转（变异成功） | 30 帧 | ❌ |
| die | 缩小→液化消失 | 30 帧 | ❌ |

### 7.2 动画原则

- 所有史莱姆动画以 **挤压与拉伸（Squash & Stretch）** 为核心
- 弹跳使用 Ease-Out-Elastic 缓动曲线
- 不使用骨骼动画实现史莱姆主体变形，优先用 Blend Shape + 代码驱动
- 环境物件不做动画（除风力摇摆等简单 shader 效果）

## 9. 风格红线（禁止事项）

1. ❌ 不使用写实贴图（照片纹理）
2. ❌ 不使用纯黑描边（Outline）
3. ❌ 不出现尖锐/恐怖/血腥元素
4. ❌ 史莱姆不使用硬边建模
5. ❌ 不使用超过 1024×1024 的贴图（Low-Poly 不需要大贴图）
6. ❌ 不使用金属/PBR 复杂材质工作流（保持简单 Lit 材质）
7. ❌ 环境不使用曲面细分
